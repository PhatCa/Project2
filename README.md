
# Manga Ecom

this site is fore people to veiw and buy manga as well as sell them on our site.


## Installation

Install my-project with npm

```bash
  npm init
  npm install express
  npm install method-override
  npm install ejs
  npm install mongoose
  npm install bcrypt
  npm install express-session
  npm install env
  npm install dotenv
```
