## Installation

Install my-project with npm

```bash
  npm init
  npm install express
  npm install method-override
  npm install ejs
  npm install mongoose
  npm install bcrypt
  npm install express-session
  npm install env
  npm install dotenv
  npm install conncect-mongo
```
